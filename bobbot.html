BoBBoT: Intelligent Assistant and Automation System

Overview: BoBBoT is a next-generation intelligent automation assistant designed by WoBBoB, built to run seamlessly on both Windows 10 and WSL (Windows Subsystem for Linux). It combines a professional GUI, deep system integration, and cutting-edge automation, all wrapped into a secure, self-updating, and fully testable bot infrastructure. Designed for power users, creators, developers, and anyone wanting to streamline their workflow, BoBBoT is your ultimate digital operator.

Key Features:

  • Modular Structure for Scalability: Core folders include gui/, core/, security/, codegen/, utils/, and testing.
  • Professional-Grade GUI (Built with PyQt6): Tabs and panels for Home, Automation, Security, System Tools, Code Generator, and Settings.
  • Automation Suite: Generate documents, send emails, launch apps, search the web, manage files.
  • Security Controls: VPN launch, role-based permissions, audit logging, DSS integration.
  • Credential Management: Secure configs, encrypted keyring support, live updates.
  • Self-Upgrading Code Generator: Template creation, backups, versioning, AI snippet support.
  • Testing and Logging: Unit testing, error tracking, test logs for all modules.
  • System Awareness: Auto-detect WSL vs Windows; smart command handling.
  • Security and Encryption: HTTPS, encrypted storage, optional 2FA, local protection.
  • Future-Ready: Plugin architecture, voice/NLP plans, deployable builds.
