BoBBoT: Intelligent Assistant and Automation System

Overview: BoBBoT is a next-generation intelligent automation assistant designed by WoBBoB, built to run seamlessly on both Windows 10 and WSL (Windows Subsystem for Linux). It combines a professional GUI, deep system integration, and cutting-edge automation, all wrapped into a secure, self-updating, and fully testable bot infrastructure.

Key Features:

  • Modular Structure with gui/, core/, security/, codegen/, utils/, testing/
  • Professional GUI with PyQt6 – multi-tabbed interface, dark/light themes
  • Automation Suite: doc creation, email, browser search, app launching
  • Security: VPN control, DSS hooks, audit logging, role-based permissions
  • Credential Management: .env/config.ini, keyring encryption, live updates
  • Self-Upgrading Code Generator with versioning and backups
  • Testing & Logging: unit tests for all modules, full logs
  • System Awareness: adjusts commands for Windows vs. WSL
  • Encryption + 2FA support for startup/login
  • Future-Ready: plugins, voice control, AI/NLP, deploy via AppImage

Typical Use Cases:

  • Automate reports, tool launching, and workflow setup
  • Secure email & file generation for personal or client work
  • VPN-protected internet tasks
  • Trigger code generation with one click
  • Daily scripting tasks via timer or command
