BoBBoT: Intelligent Assistant and Automation System

Overview: BoBBoT is a next-generation intelligent automation assistant designed by WoBBoB, built to run seamlessly on both Windows 10 and WSL (Windows Subsystem for Linux). It combines a professional GUI, deep system integration, and cutting-edge automation, all wrapped into a secure, self-updating, and fully testable bot infrastructure. Designed for power users, creators, developers, and anyone wanting to streamline their workflow, BoBBoT is your ultimate digital operator.

Key Features:

  • Modular Structure for Scalability:
    • Core folders include gui/, core/, security/, codegen/, utils/, and testing/.
    • Modules are separated by function for easy maintenance and future upgrades.
  • Professional-Grade GUI (Built with PyQt6):
    • Tabs and panels for Home, Automation, Security, System Tools, Code Generator, and Settings.
    • Dark and light themes supported with custom branding and fonts.
    • Real-time logs, action status, and system tray minimization support.
  • Automation Suite:
    • Document Creation: Generates .txt, .docx, and .pdf files using user input.
    • Email Sending: Secure SMTP email support with optional OAuth2 integration.
    • Web Search & Browser Launching: Auto-search or open pages via default browser or internal view.
    • System Program Access: Launch apps, open directories, or clean files directly from the GUI.
  • Security Controls:
    • VPN Activation: One-click VPN launch with system feedback.
    • DSS Compatibility: Integration hooks for Data Security Suites (DSS) and external firewall/antivirus.
    • Permission System: Role-based access control for sensitive tasks.
    • Audit Logging: Tracks all sensitive actions with timestamps.
  • Credential Management:
    • Uses .env or config.ini for user credentials, with support for encryption or secure keyrings.
    • API key testing built-in.
    • Settings panel allows live credential updates.
  • Self-Upgrading Code Generator:
    • Generate Python templates for new features.
    • Maintains version history and changelogs.
    • Backs up current modules before applying changes.
    • Compatible with local templates or AI-generated snippets.
  • Testing and Logging:
    • Unit tests included for GUI, automation, security, and core.
    • Logs kept for each test with pass/fail results.
    • Logging system tracks everything done by the bot: actions, errors, warnings, updates.
  • System Awareness & Cross-Platform Support:
    • Detects Windows vs WSL environment and adapts paths and permissions accordingly.
    • System commands (like open, start, sudo) auto-adjust based on detected OS.
